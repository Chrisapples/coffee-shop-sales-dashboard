# coffee-shop-sales-dashboard
Overview: Coffee shop sales analysis built using BigQuery and Tableau to evaluate revenue trends, peak transaction periods and product performance. Data sourced from Kaggle.

Tools: BigQuery, Tableau

Data Preparation: Viewed and cleaned dataset and tables 
Created transformation layer using SQL views 
Aggregated performance metrics for dashboard optimization 
Built KPI summary tables

Metrics: Total Revenue 
Total Units Sold 
Total Transactions 
Average Order Value

Insights: Revenue trends increase throughout the year during the observed period
Peak transaction activity occurs during morning hours
Product category performance highlights key revenue drivers as coffee and tea products
